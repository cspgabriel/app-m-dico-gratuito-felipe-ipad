## Política de Privacidade do Clinicafy

**Última atualização:** Junho de 2026

### 1. Quem Somos
O Clinicafy é um sistema de gestão para médicos e clínicas desenvolvido pela **AgenciAR Digital**.

**Contato:** contato@clinicafy.com.br | www.clinicafy.com.br

### 2. Dados que Coletamos

| Dado | Finalidade | Base Legal (LGPD) |
|------|------------|-------------------|
| Nome e e-mail do profissional | Autenticação e conta | Execução de contrato |
| Dados de pacientes (inseridos pelo médico) | Funcionalidade do prontuário eletrônico | Legítimo interesse (gestão clínica) |
| Dados de pagamento | Processamento de assinaturas | Execução de contrato |

### 3. Como Usamos Seus Dados
- **Autenticação:** via Google Firebase Authentication
- **Armazenamento de dados clínicos:** via MySQL hospedado na Hostinger (Brasil)
- **Pagamentos:** processados pelo Mercado Pago (não armazenamos dados de cartão)

### 4. Compartilhamento de Dados
Não vendemos seus dados. Compartilhamos apenas com:
- **Google Firebase** (autenticação)
- **Mercado Pago** (processamento de pagamentos)
- **Hostinger** (hospedagem do banco de dados)

### 5. Seus Direitos (LGPD)
- Acesso, correção e exclusão dos seus dados
- Portabilidade dos dados
- Revogação do consentimento
- Para exercer seus direitos: contato@clinicafy.com.br

### 6. Segurança
- Comunicação criptografada (HTTPS/TLS)
- Acesso restrito por autenticação
- Dados armazenados em servidores seguros

### 7. Retenção de Dados
Dados mantidos enquanto a conta estiver ativa. Após cancelamento, excluímos em até 90 dias mediante solicitação.

### 8. Contato
AgenciAR Digital | contato@clinicafy.com.br
